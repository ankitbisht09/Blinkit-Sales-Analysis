#  Blinkit Sales Analysis Dashboard

##  Objective
Analyze Blinkit sales data to identify top-performing products and revenue trends.

##  Tools Used
- SQL (PostgreSQL)
- Power BI

##  Analysis Performed
- Calculated total revenue using SQL
- Computed average order value
- Identified top 5 products by revenue
- Identified top 5 products by quantity

##  Key Insights
- Top products contribute majority of revenue
- Sales distribution is uneven across products
- High quantity does not always mean high revenue

##  Dashboard
An interactive dashboard was built using Power BI to visualize key metrics.

##  Files Included
- `queries.sql` → SQL queries used for analysis
- `blinkit.csv` → Dataset
- `dashboard.png` → Dashboard screenshot

##  Conclusion
This project demonstrates end-to-end data analysis using SQL and Power BI.
